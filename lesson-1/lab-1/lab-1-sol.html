clients
computers asking for webpages or data
servers
computers answering requests from the clients
routers
computers that are inbetween clients and servers that foward requests/responses to the correct place.
interface
the code that describes the elements on the page that interacts with the user--text, buttons, dropdowns, etcetera --along with how they look.
logic
the code tied to a web page that describes the behaviors of the web page based on how the user interacts with it
data
the dynamic information of a web page that is filled in by either the server or the client
HTML
Hypertext Markup Language; the code that nearly every web page is written in.
tag
A kind of HTML code that is wrapped around text. These are instructions to the browser on how to display the inner text. Example: 
 <h3>  A heading  </h3>
whitespace
blank spaces in your HTML document
